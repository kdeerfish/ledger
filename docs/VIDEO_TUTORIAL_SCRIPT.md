# Ledger 视频教程脚本

**时长**：约 5-8 分钟
**目标**：让新手用户快速了解如何使用 Ledger

---

## 🎬 开场（30秒）

**画面**：显示 Ledger Logo 和标题

**旁白**：
> 大家好，今天给大家介绍一个超好用的个人记账系统 - Ledger。
> 
> 它最大的特点是：**用自然语言记账**，就像跟朋友说话一样简单。
> 
> 比如你说"京东买了200块零食"，它就能自动识别出：这是支出、200块、食品酒水、商家是京东。
> 
> 不需要记命令，不需要选菜单，说话就行！

---

## 📦 安装（1分钟）

**画面**：终端界面

**旁白**：
> 首先，我们来安装 Ledger。
> 
> 最简单的方式是用 Docker，一行命令搞定：

```bash
docker run -d --name ledger -p 5800:5800 -v $(pwd)/data:/data zouzhenglu/ledger:latest
```

> 如果你是 Windows 用户，也可以下载桌面版，解压双击就能用。
> 
> 安装完成后，打开浏览器，访问 `http://localhost:5800`

**画面**：浏览器打开 Ledger 界面

---

## 📥 导入数据（1分30秒）

**画面**：浏览器界面

**旁白**：
> 如果你之前用过随手记、MoneyWiz 等记账软件，可以导入历史数据。
> 
> 首先，从你的旧软件导出 CSV 文件。
> 
> 然后，对 AI 说：

```
导入随手记的CSV
```

> AI 会告诉你：

```
请把 CSV 文件放到 data 目录，然后告诉我文件路径。
```

> 你说：

```
data/mymoney_data.csv
```

> AI 会执行导入：

```
✅ 导入完成，共 1528 笔记录
建议说"学习记账"让我分析你的记账习惯。
```

**画面**：显示导入成功的提示

---

## 🤖 AI 学习（1分钟）

**画面**：浏览器界面

**旁白**：
> 导入数据后，最重要的一步来了：**让 AI 学习你的记账习惯**。
> 
> 你只需要说四个字：

```
学习记账
```

> 你也可以说"学习ledger"、"分析我的记账数据"，AI 都能理解。
>
> AI 会分析你的所有历史数据，提取出：
> - 常用账户：微信零钱、招商银行、信用卡
> - 常用商家：拼多多、京东、美团
> - 常用类别：食品酒水、交通出行
> 
> 学习完成后，AI 会记住这些习惯，以后记账时会自动填写。

**画面**：显示学习完成的提示

---

## 📝 制作要点

### 画面要求
- **终端界面**：使用深色主题，字体清晰
- **浏览器界面**：使用 Chrome，全屏展示
- **字幕**：关键命令和提示用高亮显示

### 录制建议
- 使用 OBS Studio 录屏
- 提前准备好所有命令和数据
- 录制时放慢语速，给观众时间阅读
