## 1. 什么是 Ledger

Ledger 是一个**个人记账系统**，它有三个特点：

- ✅ **AI 助手**：用自然语言就能记账，比如"京东买了200块零食"
- ✅ **多端使用**：手机、电脑、NAS 都能用
- ✅ **数据安全**：数据存在你自己的设备上，不上传到任何服务器

## 2. 安装 Ledger

### 方法一：Docker 安装（推荐）

**适用场景**：NAS、服务器、任何有 Docker 的设备

```bash
# 1. 拉取镜像
docker pull zouzhenglu/ledger:latest

# 2. 启动服务
docker run -d \
  --name ledger \
  -p 5800:5800 \
  -v $(pwd)/data:/data \
  --restart unless-stopped \
  zouzhenglu/ledger:latest

# 3. 访问 Ledger
# 打开浏览器，访问 http://localhost:5800
```

**国内用户加速**：
```bash
docker pull crpi-1bkinvfgt16i5pgx.cn-shenzhen.personal.cr.aliyuncs.com/deerfish/ledger:latest
```

### 方法二：Windows 桌面版

**适用场景**：Windows 用户，不想用 Docker

1. 访问 [GitHub Releases](https://github.com/kdeerfish/ledger/releases)
2. 下载 `ledger-windows.zip`
3. 解压后双击 `ledger.exe`
4. 浏览器自动打开 `http://localhost:5800`

### 方法三：源码安装

**适用场景**：开发者，想自己编译

```bash
# 1. 克隆项目
git clone https://github.com/kdeerfish/ledger.git
cd ledger

# 2. 安装依赖
pip install -e .

# 3. 启动服务
python web/run.py

# 4. 访问 http://localhost:5800
```

## 3. 首次使用：导入历史数据

如果你之前用过**随手记、MoneyWiz** 等记账软件，可以导出 CSV 后导入到 Ledger。

### 步骤 1：导出 CSV 文件

从你的旧记账软件导出 CSV 文件，确保包含以下列：

| 列名 | 必填 | 说明 |
|------|------|------|
| 交易类型 | ✅ | 支出/收入 |
| 日期 | ✅ | YYYY/MM/DD HH:MM |
| 金额 | ✅ | 数字 |
| 类别 | ✅ | 食品酒水、交通等 |
| 子类别 | 可选 | 零食、早午餐等 |
| 账户 | 可选 | 微信零钱、招商银行 |
| 商家 | 可选 | 京东、拼多多 |

### 步骤 2：上传 CSV 文件到服务器

**Docker 用户**：
```bash
# 把 CSV 文件放到 NAS 的 data 目录
# 例如：/volume1/docker/ledger/data/mymoney_data.csv
```

**本地用户**：
```bash
# 直接放在项目目录下即可
```

### 步骤 3：执行导入命令

**Docker 用户**：
```bash
# 在 NAS 终端执行
docker exec -it ledger python scripts/import_ledger.py /data/mymoney_data.csv
```

**本地用户**：
```bash
python scripts/import_ledger.py data/mymoney_data.csv
```

**导入成功后**：
```
✅ 导入完成，共 1528 笔记录
```

## 4. 让 AI 学习你的记账习惯

**这一步很重要！** 让 AI 学习你的历史数据，以后记账时它会自动填写账户、商家、类别等信息。

### 如何触发学习

**直接对 AI 说**：
```
学习记账
```

就这么简单，四个字就行！

> 💡 **提示**：你也可以说"学习ledger"、"分析我的记账数据"等，AI 都能理解。

### AI 会做什么

1. **分析你的数据**：读取所有历史记录
2. **提取习惯模式**：
   - 常用账户：微信零钱、招商银行、信用卡
   - 常用商家：拼多多、京东、美团
   - 常用类别：食品酒水、交通出行
   - 家庭成员：本人、家人
3. **保存到记忆**：以后记账时自动参考

### 学习后的效果

**学习前**，你需要说：
```
记一笔，支出 200 元，类别食品酒水，子类别零食，账户信用卡，商家京东
```

**学习后**，你只需要说：
```
京东买了200块零食
```

AI 会自动填写：
- ✅ 类别：食品酒水
- ✅ 子类别：零食
- ✅ 账户：信用卡（你的常用账户）
- ✅ 商家：京东

> 💡 **提示**：学习后记忆永久生效，无需重复学习。如果数据变化较大，可以说"重新学习记账"。

## 5. 日常记账

### 用自然语言记账

**示例 1：买零食**
```
记一笔，拼多多买了6块8的零食
```

**示例 2：打车**
```
今天打车花了20块
```

**示例 3：收到工资**
```
记一笔收入，工资5000到招商银行
```

**示例 4：网购**
```
京东买了双鞋，300块，用信用卡付的
```

### AI 会自动处理

- ✅ 自动识别**支出/收入**
- ✅ 自动填写**类别**（食品酒水、交通等）
- ✅ 自动填写**账户**（你的常用账户）
- ✅ 自动填写**商家**（如果提到的话）
- ✅ 自动使用**当前时间**（也可以指定日期）

### 指定日期记账

```
记一笔，昨天打车花了30
记一笔，2026年6月15号买了本书50块
```

## 10. 常见问题

### Q1：导入 CSV 失败怎么办？

**检查以下几点**：
1. CSV 文件编码是否为 UTF-8
2. 第一行是否为表头
3. 必填列是否都有（交易类型、日期、金额、类别）
4. 日期格式是否正确（YYYY/MM/DD HH:MM）

### Q2：学习后 AI 还是记不住我的习惯？

**解决方案**：
1. 确保历史数据足够多（至少 100 条以上）
2. 重新说"学习记账"或"重新学习记账"让 AI 重新分析
3. 手动告诉 AI 你的习惯：
   ```
   我一般用微信零钱买早餐
   我打车都用滴滴
   ```

### Q3：数据存在哪里？

**Docker 用户**：
- 数据文件：`./data/ledger.db`
- 备份时复制这个文件即可

**本地用户**：
- 数据文件：项目根目录下的 `ledger.db`

### Q4：如何备份数据？

```bash
# Docker 用户
cp ./data/ledger.db ./data/ledger_backup_$(date +%Y%m%d).db

# 本地用户
cp ledger.db ledger_backup_$(date +%Y%m%d).db
```

### Q5：如何迁移到新设备？

1. 备份 `ledger.db` 文件
2. 在新设备安装 Ledger
3. 将 `ledger.db` 放到对应目录
4. 启动服务即可

### Q6：AI 记错了怎么办？

**修改记录**：
```

把第42条记录的类别改成交通
```

