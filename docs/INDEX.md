## 🤝 贡献文档

如果你想改进文档，欢迎提交 Pull Request！

### 如何贡献

1. Fork 本仓库
2. 创建你的分支 (`git checkout -b docs/improve-getting-started`)
3. 修改文档
4. 提交修改 (`git commit -m 'Improve getting started guide'`)
5. 推送到分支 (`git push origin docs/improve-getting-started`)
6. 打开一个 Pull Request

### 文档规范

- 使用 Markdown 格式
- 保持简洁明了
- 添加示例代码
- 添加截图（如果需要）
- 检查链接是否正确
