## 🎯 快速上手

1. **启动 Ledger**：`docker run -d -p 5800:5800 -v $(pwd)/data:/data zouzhenglu/ledger`
2. **打开浏览器**：访问 `http://localhost:5800`
3. **导入数据**：`导入随手记的CSV`
4. **学习习惯**：`学习`
5. **开始记账**：`记一笔，买了杯咖啡15块`

---

**💡 提示**：第一次使用建议先导入历史数据，然后说"学习记账"让 AI 了解你的记账习惯。
